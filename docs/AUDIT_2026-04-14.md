## 🔍 Findings Détaillés

### 1. **Upload de Documents** (Audit-001)
✅ **Status:** COMPLIANT

**Fichier concerné:** `app/Http/Controllers/Admin/DocumentController.php`

**Mécanismes de sécurité :**
- Validation via `DocumentRequest` (FormRequest)
- MIME types whitelist : PDF, DOC, DOCX, XLSX, XLS, TXT, JPG, JPEG, PNG
- Limite de taille : 10 MB (10240 KB)
- Autorisation : `isAdmin()` required
- Stockage : `storage_path('app/documents')` — Private disk
- Sanitisation du filename : Suppression des caractères de contrôle (x00-x1F, x7F) + 200 chars max

**Code exemple (download) :**
```php
$raw = $document->original_name ?? basename($document->path);
$filename = preg_replace('/[\\x00-\\x1F\\x7F]+/', '', (string) $raw);
$filename = trim($filename) ?: basename($document->path);
if (mb_strlen($filename) > 200) {
    $filename = mb_substr($filename, 0, 200);
}
```

**Verdict :** ✅ **Bien implémenté.** La sanitisation des filenames est robuste.

**Verdict :** ✅ **OK. Intentionnel et bien segmenté.**
